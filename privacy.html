Privacy Policy

Last Updated: February 2026

1. Information We Collect

XAYTHEON is designed to help you track GitHub growth. To provide our services, we collect:

  • GitHub Data: Public profile information, contribution history, and repository metadata accessed via the GitHub API.
  • Usage Data: Information on how you interact with our analytics tools and 3D visualizations.
  • Preferences: Theme settings and language selections stored locally on your device.

2. How Data is Handled

Your privacy is paramount. We handle your data according to these principles:

  • We do not store your GitHub passwords or private authentication tokens on our servers.
  • Analytics are processed locally or via secured API calls to provide real-time insights.
  • Personalization data (like dark mode preference) stays in your browser's local storage.

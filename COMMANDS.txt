## üß™ Testing Commands

### In Browser Console (F12)

**View all favorites:**
```javascript
localStorage.getItem('favoriteRepositories')
```

**Pretty print favorites:**
```javascript
console.table(JSON.parse(localStorage.getItem('favoriteRepositories') || '[]'))
```

**Get favorite count:**
```javascript
JSON.parse(localStorage.getItem('favoriteRepositories') || '[]').length
```

**Add a test favorite:**
```javascript
window.favoritesManager.addFavorite({
  id: 999,
  name: 'test-repo',
  owner: 'testuser',
  url: 'https://github.com/testuser/test-repo',
  stars: 100,
  language: 'JavaScript'
})
```

**Clear all favorites:**
```javascript
window.favoritesManager.clearAll()
```

**Get all favorites:**
```javascript
window.favoritesManager.getFavorites()
```

**Check if specific repo is favorited:**
```javascript
window.favoritesManager.isFavorited(123)
```
